In order to make a release, do

mate lib/knights_tour.rb                # update version number
mate CHANGELOG.rdoc                     # describe changes
rake debug_gem > knights_tour.gemspec   # generate a gemspec
mate knights_tour.gemspec               # remove the first line
git diff                                # review the changes
git commit -am "Bump to release-<version>."
git tag release-<version>
git push
git push --tags
