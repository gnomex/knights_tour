= Knight's Tour

A program that attempts to find a solution to the {Knight's Tour problem}[http://en.wikipedia.org/wiki/Knight%27s_Tour].

== Installation

Use git to get the latest version:

  $ git clone git://github.com/tuomas/knights_tour.git

== Usage

Change the working directory of the command line shell into the project's root directory and enter

  $ ./bin/knights_tour

For options, see

  $ ./bin/knights_tour -h
